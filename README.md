# Snake and Ladder Game (Python)

## 📌 Overview
A multiplayer Snake and Ladder game built using Python Turtle graphics.

## 🎮 Features
- Supports 2–4 players
- Dice roll simulation
- Snakes and ladders logic
- Interactive gameplay
- Graphical board using turtle

## ⚙️ Tech Stack
- Python
- Turtle Graphics

## 🚀 How to Run
```bash
python snake_ladder_game.py
